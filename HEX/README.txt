If your engine is 6 cylinders, install SparkEFI-V1.1-speeduino-202402.2-6+3.hex software.
If the Cam sensor is connected, this file will operate the fuel injectors in full sequential and the ignition coils in semi sequential.
If desired, both injector and ignition can be made semi-sequential via Tunerstudio.

If your engine is 4 cylinders, install SparkEFI-V1.1-speeduino-202402.2-5+4.hex software.
This software will be able to run your engine in both Full sequential and semi sequential modes.
